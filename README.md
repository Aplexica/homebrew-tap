# Aplexica Homebrew Tap

Official Homebrew tap for Aplexica.

## Install

```sh
brew install Aplexica/tap/aplexica
```

After installation, run:

```sh
aplexica setup
```

The formula installs the `aplexica` CLI/daemon and helper binaries used by the
local desktop experience. Release automation in `Aplexica/Aplexica` maintains
the formula for tagged releases.
